# Pages

## `/`
- 홈 화면
- 현재 활성/비활성 기능 상태를 안내
- 각 주요 페이지로 이동 가능

## `/dart`
- 실시간 DART 기능 비활성화 안내 페이지

## `/dart/opendart-fast`
- OPEN DART 빠른 공시 기능 비활성화 안내 페이지

## `/sec`
- 실시간 SEC 공시 기능 비활성화 안내 페이지

## `/scanners`
- 국내 스캐너 기능 비활성화 안내 페이지

## `/scanners/us`
- 미국 스캐너 기능 비활성화 안내 페이지

## `/watchlist`
- 관심 종목 관리 페이지

## `/notifications`
- 알림 센터 페이지
